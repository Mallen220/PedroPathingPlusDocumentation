## What is Pedro Pathing Plus?

Pedro Pathing Plus is an advanced software library and visualization toolset designed specifically for **FIRST Tech Challenge (FTC)** teams. It provides a robust, highly optimized algorithm for autonomous path planning, trajectory generation, and robot navigation. By simplifying the complexities of odometry and real-time movement execution, Pedro Pathing Plus enables teams to execute highly complex, reliable autonomous routines.

Coupled with the **Pedro Pathing Visualizer**, teams can intuitively draw paths, simulate robot movements, and directly export ready-to-use Java code (both standard OpMode and Command-Based), drastically reducing the time required to program an FTC robot's autonomous period.
